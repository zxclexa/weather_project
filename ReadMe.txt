Прогнозер Погоды(Виджет погоды)

Прогнозер погоды - очень простая программа которая показывает погоду в выбранном вами городе на данный момент времени.Присутствует английский язык.

Проект сделан на Python 3.10.Все нужные для работы библиотеки есть в файле requirements.txt.Их можно установить через cmd командой pip install requirements.txt или pip3 install requirements.txt

Чтобы запустить программу нужно нажать на файл project_weather_main_2.exe в папке dist -> project_weather_main_2.
